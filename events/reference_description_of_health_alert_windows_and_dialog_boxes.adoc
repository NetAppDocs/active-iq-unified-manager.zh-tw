= 警示視窗和對話方塊說明
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您應該使用「新增警示」對話方塊來設定警示、以接收有關事件的通知。您也可以從「警示設定」頁面檢視警示清單。
