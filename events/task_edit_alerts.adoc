= 編輯警示
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以編輯警示內容、例如警示所關聯的資源、事件、收件者、通知選項、通知頻率、 及相關指令碼。

*您需要的是什麼*

您必須具有應用程式管理員角色。

.步驟
. 在左導覽窗格中、按一下*儲存管理*>*警示設定*。
. 在*警示設定*頁面中、選取您要編輯的警示、然後按一下*編輯*。
. 在*編輯警示*對話方塊中、編輯名稱、資源、事件和動作區段、 視需要而定。
+
您可以變更或移除與警示相關的指令碼。

. 按一下「 * 儲存 * 」。
