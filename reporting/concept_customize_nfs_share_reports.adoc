= 自訂NFS共用報告
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以自訂NFS共用報告、以分析儲存系統上磁碟區的NFS匯出原則和規則相關資訊。例如、您可以自訂報告、以預設匯出原則顯示具有無法存取掛載路徑和磁碟區的磁碟區。
