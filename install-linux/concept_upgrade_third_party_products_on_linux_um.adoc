= 安裝 Unified Manager 後升級第三方產品
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當Unified Manager已安裝在Linux系統上時、您可以升級第三方產品、例如JRE。

開發這些協力廠商產品的公司會定期報告安全性弱點。您可以依照自己的排程升級至此軟體的較新版本。
