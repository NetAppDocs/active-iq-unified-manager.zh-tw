= 刪除註釋
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以刪除不再需要的自訂註釋及其值。

*您需要的是什麼*

* 您必須具有應用程式管理員或儲存管理員角色。
* 註釋值不得用於其他註釋或群組規則。


.步驟
. 在左導覽窗格中、按一下*儲存管理*>*註釋*。
. 在*註釋*標籤中、選取您要刪除的註釋。
+
隨即顯示所選附註的詳細資料。

. 按一下「*動作*」>「*刪除*」以刪除選取的註釋及其值。
. 在警告對話方塊中、按一下*是*以確認刪除。
