= 管理使用者存取
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以建立角色並指派功能、以控制使用者對Active IQ Unified Manager 功能的存取。您可以識別具備必要功能的使用者、以便在Unified Manager中存取所選物件。只有具備這些角色和功能的使用者、才能在Unified Manager中管理物件。
